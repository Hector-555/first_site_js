HTML

Язык разметки веб-страниц. Результат интерпретируется браузером, и пользователь видит текст на экране в заданном разработчиком виде. Особенность языка разметки HTML заключается в преобразовании простых команд в визуальные объекты. Основные элементы HTML: форматирование текста, текстовые блоки, таблицы, вставка объектов, гиперссылки.

CSS

Они же "каскадные таблицы стилей" используются для визуального оформления веб-сайтов. CSS отвечает за то, как выглядят объекты, размещенные с помощью HTML: размер, цвет, фоновое изображение, степень прозрачности, расположение относительно других элементов, поведение при наведении курсора, визуальное изменение кнопок при нажатии и т.п. Практически любые «внешние проявления» сайта создаются с помощью CSS, не прибегая к использованию более сложных языков программирования. С помощью CSS даже можно рисовать, создавать объекты любых форм, анимировать их и т.п. Одна из ключевых задач CSS в современной верстке – создание макетов сайта.

JS

Использование скриптов JavaScript обусловлено желанием добавить функции, которых нет и не будет в HTML/CSS. Например, обработку ввода контактов (отправка на указанный email, выдача сообщений, автоматический обратный звонок). За счет мультипарадигменности язык JS способен выполнять разнотипные операции: математические вычисления, обработку и валидацию данных в HTML-формах, взаимодействие с пользователем, взаимодействие с HTML-элементами на странице, графические эффекты, анимацию. Пользователь совершил некоторое действие -- Браузер зафиксировал совпадение неких условий -- Запустился JavaScript-код, «привязанный» к событию -- Страница обновилась полностью или частично
